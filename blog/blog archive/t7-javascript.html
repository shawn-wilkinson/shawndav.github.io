Ruby and JavaScript

Differences in If...Else Statements

May 24, 2015

Until just recently, my studies at Dev Bootcamp have focused on Ruby, a popular object-oriented programming language. This past week, we've been introduced to JavaScript. Although Ruby and JavaScript are alike in many ways, they also have some major differences. In this blog post, I'd like to explore some of these differences by comparing how Ruby and JavaScript address if...else statements.

Below you will see two if...else statements. They are identical except one is written in JavaScript and the other is in Ruby. Both are written to compare a user's score with an existing high score to determine if a new record has been set.

Ruby:
user_score = 20
high_score = 10
if user_score > high_score
    puts "Amazing! You set a new high score!"
elsif user_score == high_score
    puts "Great job! You tied the high score!"
else
    puts "Good job!"
end

JavaScript:
var userScore = 20
var highScore = 10
if (userScore > highScore) {console.log("Amazing! You set a new high score!")}
else if(userScore === highScore){console.log("Great job! You tied the high score!")}
else{console.log("Good job!")};

As you can see above, both Ruby and JavaScript utilize a similar structure. They both use if and else, while Ruby uses elsif and JavaScript uses else if. Beyond this, there are a number of differences.

This first difference can be seen in the statement that is being evaluated. In ruby if user_score > high_score is used, while in JavaScript If(userScore > highScore) is used. JavaScript requires parenthesis to be used to contain the evaluated statement. After the statement is evaluated, the code block is also formatted differently in each of the two languages. In Ruby, the code between the line with the if and the line with the elsif will run in the event the if statement's condition is true. This code must be on a new line. In JavaScript this code does not need to be on a new line, but must be written inside curly brackets. To finish an if...else statement in Ruby an end is required, while in JavaScript only a semicolon is used.
